# ☕ Café Animado

Uma página simples e moderna criada com **HTML5** e **CSS3**, apresentando uma animação interativa de uma caneca de café que enche lentamente ao passar o mouse.  
O projeto foi desenvolvido utilizando o auxílio do Google AI Studio para geração da ideia inicial e estrutura do código.

---

## 📌 Sobre o Projeto

Este projeto tem como objetivo demonstrar o uso de animações modernas em CSS puro, criando uma experiência visual simples, elegante e interativa.

Ao posicionar o cursor sobre a caneca, o café sobe lentamente simulando o enchimento da bebida.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Animações com `@keyframes`
- Transições CSS
- Hover Effects

---

## 🎨 Funcionalidades

- ☕ Caneca estilizada com CSS
- ✨ Animação suave de preenchimento
- 🖱️ Interação ao passar o mouse
- 📱 Layout simples e moderno
- ⚡ Projeto leve e responsivo

---

## 🤖 Prompt Utilizado

```txt
crie uma pagina simples com uma animçao de caneca de cafe que enche lentamente utilizando apenas CSS e HTML moderno ao passar o mouse.
```

2. Abra o arquivo `index.html` no navegador

## 📸 Prévia do Projeto

Uma animação minimalista de uma caneca de café com efeito de preenchimento suave utilizando apenas CSS.

---

## 💡 Objetivo

Este projeto foi criado para praticar:

- Estruturação em HTML
- Estilização moderna com CSS
- Animações e transições
- Interatividade sem JavaScript
